The code run_analysis.R must be run in the 'UCI HAR Dataset' directory once the file:
https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip
is unzipped.

It takes the data from test and train, merges it and extract only the data from mean and std values. It stores it in 'final¡ data frame.

It also calculates the mean for each activity and subject of the former data and stores it in resul data frame.
